🚀 خطوات التثبيت السريعة
───────────────────────────────────────────────────────────────────

1. التحضير (محلياً):
   composer install --no-dev --optimize-autoloader
   cp .env.ipage .env
   # عدّل معلومات قاعدة البيانات في .env

2. إنشاء قاعدة البيانات (iPage cPanel):
   MySQL Databases → Create Database
   Add User → Grant Privileges

3. رفع الملفات (FTP/File Manager):
   laravel-app/ → /home/username/laravel-app/
   public files → /home/username/public_html/

   اقرأ FILES_TO_UPLOAD.txt للتفاصيل

4. ضبط الصلاحيات:
   chmod -R 755 storage/
   chmod -R 755 bootstrap/cache/

5. تهيئة قاعدة البيانات:
   # عبر SSH أو استيراد SQL عبر phpMyAdmin
   php artisan migrate --force
   php artisan make:user

6. الوصول:
   🌐 الموقع: https://yourdomain.com
   🔐 الإدارة: https://yourdomain.com/admin
   🛠️  الصيانة: https://yourdomain.com/admin/maintenance

───────────────────────────────────────────────────────────────────
🆘 حل المشكلات السريع
───────────────────────────────────────────────────────────────────

❌ خطأ 500
   → تحقق من صلاحيات storage/ و bootstrap/cache/
   → راجع معلومات قاعدة البيانات في .env
   → تأكد من وجود APP_KEY

❌ صفحة بيضاء
   → تأكد من وجود vendor/
   → تحقق من صحة .env
   → افحص storage/logs/laravel.log

❌ 404 على الصفحات
   → تأكد من وجود .htaccess في public_html/
   → تحقق من تفعيل mod_rewrite

❌ الستايلات لا تظهر
   → راجع APP_URL في .env
   → تأكد من رفع جميع ملفات public/
